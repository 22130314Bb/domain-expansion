🔮 — Cursed Technique Visualizer

> “Throughout heaven and earth, I alone am the honored one.”



SAT0RU é um visualizador de técnicas amaldiçoadas em tempo real, inspirado diretamente em Jujutsu Kaisen 呪術廻戦.
Usando gestos da mão, webcam e partículas 3D, o projeto recria Expansões de Domínio e Técnicas Amaldiçoadas como se vc fosse um feiticeiro de verdade.

Construído com MediaPipe + Three.js 🧠




---

✨ O que esse projeto faz?

O projeto detecta gestos da mão pela webcam e transforma isso em técnicas amaldiçoadas renderizadas com partículas volumétricas em 3D.

Cada gesto ativa uma técnica diferente, tudo em tempo real, direto no navegador.


---

👉 Polegar + Indicador se tocando.

---

🧪 Tecnologias Usadas

Three.js — Renderização 3D e partículas

MediaPipe Hands — Detecção de mãos e gestos

JavaScript — O caos organizado

WebGL — Performance em tempo real


Tudo roda direto no navegador, sem instalar nada pesado.


---

🚀 Como rodar o projeto

Pré-requisitos

Navegador moderno (Chrome, Edge ou Firefox)

Webcam funcionando


Instalação

1. Clone o repositório

git clone https://github.com/brunogustavo74/domain-expansion.git
cd domain-expansion


2. Rodar o projeto

No VS Code:

Instale a extensão Live Server

Clique com o botão direito em index.html

Selecione “Open with Live Server”



3. Permita acesso à webcam

Sem câmera = sem feitiço, vacilão





---

⚠️ Avisos Importantes

Iluminação ruim pode ferrar a detecção dos gestos

Movimentos muito rápidos podem não ser reconhecidos

Isso é um experimento visual, não um jogo fechado (ainda 👀)
